# Christopher Carazas Art-Book Website · V6.6

## Preview on Mac
1. Unzip the package.
2. Double-click `START-PREVIEW.command`.
3. Open `http://127.0.0.1:8161/?build=v6.6`.

## Production build
The site remains static and GitHub Pages compatible. Shared navigation, footer, plate metadata, schema, conversion modules, and selected content pages are generated by `scripts/build_site.py` from the canonical publication ledger in `data/site.json`.

Useful commands:

```bash
python3 scripts/build_site.py
node scripts/build_css.cjs
python3 scripts/build_sitemaps.py
python3 scripts/audit_site.py
```

## Substack feed
A scheduled GitHub Action refreshes `data/substack-posts.json`. The last valid cache and the editorial HTML cards remain visible if the feed is unavailable.
